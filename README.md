# ShopEasy - E-Commerce Shopping Website

## Project Overview
ShopEasy is a full-stack e-commerce shopping website developed using React, Flask, and SQLite. The application allows users to browse products, search and filter items, add products to cart, make fake payments, and manage products through an admin dashboard.

---

## Features

- User Registration and Login
- Protected Routes Authentication
- Product Search and Category Filter
- Add to Cart Functionality
- Quantity Management
- Fake Payment Gateway
- Toast Notifications
- Admin Dashboard
- Add Products Dynamically
- Upload Product Images
- Responsive UI Design

---

## Technologies Used

### Frontend
- React JS
- React Router DOM
- Axios
- React Toastify
- CSS

### Backend
- Flask
- Flask CORS
- SQLite3
